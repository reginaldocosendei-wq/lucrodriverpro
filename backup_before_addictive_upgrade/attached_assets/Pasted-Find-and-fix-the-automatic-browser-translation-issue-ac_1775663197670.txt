Find and fix the automatic browser translation issue across the entire app.

The problem:
Google Chrome is auto-translating parts of the page, changing words and breaking the UI and functionality. Some texts appear, disappear, or change after refresh because browser translation is modifying the DOM.

Apply these fixes carefully across the project:

1. In the main HTML document, add this inside <head>:
<meta name="google" content="notranslate">
<meta http-equiv="Content-Language" content="pt-BR">

2. Make sure the root html tag uses Brazilian Portuguese:
<html lang="pt-BR">

3. Add protection against translation in the main app container and critical UI elements by using:
translate="no"

- status badges
- counters
- tabs

- summary widgets
- form labels that must remain stable
