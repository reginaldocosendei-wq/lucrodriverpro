Implement a real payment system in my app "Lucro Driver" using Stripe for subscriptions and prepare support for PIX (Brazil).

---

## 🎯 GOAL

Allow users to subscribe to PRO and pay monthly or yearly.

---

## 💳 STRIPE SETUP

Use Stripe Checkout for subscriptions.

Create plans:

- Monthly Plan: R$19.90
- Yearly Plan: R$149.90

---

## 🔐 BACKEND (NODE.JS)

Install Stripe:

npm install stripe

Create a Stripe instance:

const stripe = require('stripe')(process.env.STRIPE_SECRET_KEY);

---

## 💰 CREATE CHECKOUT SESSION

Create endpoint:

POST /create-checkout-session

- Receive userId
- Create Stripe session with:
  - Mode: subscription
  - Price: monthly or yearly
  - Success URL: /success
  - Cancel URL: /cancel

Return session URL

---

## 📱 FRONTEND BUTTON

When user clicks "Upgrade to PRO":

- Call backend
- Redirect to Stripe Checkout page

---

## ✅ SUCCESS PAGE

After payment:

- Show:
"Payment successful! PRO activated."

---

## 🔄 WEBHOOK (IMPORTANT)

Create endpoint:

POST /webhook

Listen to:

- checkout.session.completed

When triggered:

- Get userId from metadata
- Update database:

isPro = true
trialActive = false

---

## 🔒 ACCESS CONTROL

If isPro = true:
→ Unlock all PRO features

If false:
→ Show paywall

---

## 🧾 DATABASE

Store:

- userId
- stripeCustomerId
- subscriptionId
- isPro

---
